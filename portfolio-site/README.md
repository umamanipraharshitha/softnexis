# Portfolio Site

A simple 3-page portfolio website with a clean, modern UI.

## Project Structure

```text
portfolio-site/
  index.html      — Home page
  about.html      — About page
  contact.html    — Contact page
  styles/
    main.css      — Global styles
  images/         — Store all images
  README.md       — Project documentation
```

## Features

- 3 pages with consistent navigation (Home, About, Contact)
- Responsive layout
  - Mobile (≤768px): stacked content + hamburger menu
  - Desktop (>768px): multi-column sections
- Modern CSS: Flexbox, CSS Grid, media queries
- Hover effects on links, buttons, and cards

## How to Run

1. Open the `portfolio-site` folder.
2. Double-click `index.html`, or use Live Server in VS Code/Cursor.
3. Resize the browser to test mobile vs desktop layout.

## Tech Stack

- HTML5
- CSS3 (Flexbox + Grid)
